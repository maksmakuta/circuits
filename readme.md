# Circuits

 Simple app to create, edit and simulate logic circuits
 
## Technologies used:

 - C++20
 - CMake 4.0
 - OpenGL 4.6 Core
 - SDL3
 - GLM
 - Freetype2
 - GLAD (already included)
 - lodePNG (already included)
